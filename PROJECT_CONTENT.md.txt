# FLAGSHIP PROJECT

## Project Name
Design and Implementation of a Spherical Workspace Robotic Arm for Spray Painting

---

## Project Category
Robotics Engineering / Manipulator Design / Embedded Systems / Motion Planning

---

## Project Type
Team Project

---

## Short Recruiter Summary

A 5-DoF spherical workspace robotic arm designed for automated spray painting applications, integrating robotic kinematics, trajectory planning, embedded control, and browser-based operation.

---

## Problem Statement

Manual spray painting operations often lead to:
- inconsistent coating
- paint wastage
- reduced precision
- worker exposure to hazardous chemicals

Industrial environments require robotic systems capable of:
- repeatable motion
- smooth trajectory execution
- precise end-effector positioning
- uniform coating

---

## Project Objective

The goal of this project was to design and develop a robotic arm capable of performing controlled spray painting operations using:
- spherical workspace motion
- smooth trajectory planning
- accurate end-effector control
- lightweight mechanical design
- embedded servo control

---

## Engineering Features

### Mechanical System

- optimized reach and flexibility

### Kinematics

### Control Interface
- browser-based control system

### Fluid System
- compressor integration

---

## Key Technical Specifications

### Workspace
- spherical workspace
- radial reach: approximately 400–430 mm
- vertical sweep: 0°–135°
- base rotation: 0°–360°

### Link Lengths
- Base Link: 140 mm
- Shoulder Link: 116.68 mm
- Arm Link: 88.04 mm
- Wrist Link: 15 mm
- End Effector / Nozzle: 75 mm

### Motors Used
#### MG996R Servo
Used for:
- base
- shoulder
- arm joints

#### MG90S Servo
Used for:
- wrist
- end-effector orientation

### Electronics
- Arduino UNO
- Buck Converter
- Bluetooth Module
- 12V 10A SMPS
